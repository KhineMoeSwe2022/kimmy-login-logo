# Kimmy's Custom Login Image and URL

This plugin lets you customize the WordPress login screen by replacing the default logo with your own image and setting a custom link URL for the logo.


## Documentation

1. Download the plugin zip file
2. Go to Plugins > Add New > Upload in the WordPress dashboard
3. Select the plugin zip file and click Install Now
4. Activate the plugin through the 'Plugins' menu in WordPress

Congratulations! You have successfully customized your WordPress login logo using the Kimmy Login Logo plugin.
